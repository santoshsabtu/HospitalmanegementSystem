# Hospital Management System

## 🔹 Description
A Java-based Hospital Management System using MySQL database to manage patients, doctors, and appointments.

## 🔹 Features
- Add Patient
- View Patient
- Add Doctor
- Book Appointment
- MySQL Database Integration

## 🔹 Technologies Used
- Java
- MySQL
- JDBC
- IntelliJ IDEA

## 🔹 Database Tables
- patients
- doctors
- appointments

## 🔹 How to Run
1. Setup MySQL database
2. Update DB credentials in code
3. Run Main.java
